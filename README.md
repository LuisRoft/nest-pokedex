# Ejecutar en desarrollo

1. Clonar repositorio
2. Instalar dependencias

```ssh
pnpm install
```

3. Tener Nest CLI instalado

```ssh
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```ssh
docker-compose up -d
```

5. Levantar el proyecto en desarrollo y en modo watch

```ssh
pnpm start:dev
```

## Stack usado

- MongoDB
- Nest
